<h3>Inheritance dalam Java: Membuat Kelas Turunan</h3><figure><img alt="" src="https://cdn-images-1.medium.com/max/1024/0*j228EAaH0-pIXt2y.jpg" /></figure><p>Inheritance atau pewarisan adalah salah satu konsep dasar dalam Pemrograman Berorientasi Objek (PBO). Konsep ini memungkinkan sebuah kelas untuk mewarisi atribut dan metode dari kelas lain. Dalam tutorial ini, kita akan membahas apa itu inheritance, bagaimana mengimplementasikannya dalam Java, serta melakukan studi kasus sederhana mengenai sistem kendaraan.</p><h4>1. Apa itu Inheritance?</h4><p>Inheritance memungkinkan kita untuk membuat kelas baru berdasarkan kelas yang sudah ada. Kelas baru tersebut akan mewarisi atribut dan metode dari kelas induk (parent class). Hal ini memudahkan pengelolaan kode karena kita bisa menggunakan kembali kode yang ada tanpa perlu menulis ulang.</p><p>Contoh: Kelas Mobil dan SepedaMotor mewarisi atribut dari kelas Kendaraan.</p><h4>2. Keuntungan Menggunakan Inheritance</h4><ul><li><strong>Reusability</strong>: Kode dari kelas induk dapat digunakan kembali oleh kelas turunan.</li><li><strong>Maintenance</strong>: Perubahan pada kelas induk akan otomatis mempengaruhi kelas turunannya.</li><li><strong>Extensibility</strong>: Kelas turunan bisa menambahkan fungsionalitas baru tanpa mengubah kelas induk.</li></ul><h4>3. Kapan Menggunakan Kelas Turunan?</h4><p>Kelas turunan digunakan ketika beberapa kelas memiliki atribut dan metode yang sama, sehingga mereka dapat berbagi kode melalui pewarisan dari kelas induk. Misalnya, Mobil dan SepedaMotor adalah dua jenis kendaraan yang memiliki kecepatan dan nama, sehingga masuk akal untuk membuat kelas Kendaraan sebagai kelas induk mereka.</p><h3>Persiapan</h3><p>Sebelum kita mulai, pastikan Anda telah menginstall <strong>NetBeans </strong>caranya klik <a href="https://medium.com/@dikaelsaputra/cara-instal-netbeans-ide-di-windows-7e29e0815459">di sini</a>. Berikut adalah langkah-langkah awal:</p><ul><li><strong>Buka NetBeans</strong> yang sudah terinstall di komputer Anda.</li><li><strong>Buat proyek baru</strong> dengan nama PraktikumPBO_5.</li></ul><figure><img alt="" src="https://cdn-images-1.medium.com/max/827/1*xziUPrw9kk_IxP5-ErF9_A.png" /></figure><figure><img alt="" src="https://cdn-images-1.medium.com/max/827/1*BDg6lPRd_wyWhkXAZFotYA.png" /></figure><figure><img alt="" src="https://cdn-images-1.medium.com/max/827/1*typP8ApSABtxeGxYLsbO5A.png" /></figure><ul><li><strong>Buat package baru</strong> dengan nama praktikum5.</li></ul><figure><img alt="" src="https://cdn-images-1.medium.com/max/827/1*Sg-q8qxVUr0n4_Cy-BjGiw.png" /></figure><figure><img alt="" src="https://cdn-images-1.medium.com/max/827/1*rYNYA0-TPEd9WOEN-giekQ.png" /></figure><h3>Membuat Kelas Turunan Kendaraan</h3><p>Kita akan mulai dengan membuat kelas induk dan dua kelas turunannya.</p><h4>1. Pewarisan: Kelas Kendaraan, KendaraanDarat, Mobil, dan SepedaMotor</h3><p>Dalam hierarki ini, kita menambahkan satu kelas menengah, yaitu KendaraanDarat, yang menjadi induk bagi Mobil dan SepedaMotor.</p><h4>Kelas Induk: Kendaraan.java</h4><pre>public class Kendaraan {<br />    String nama;<br />    int kecepatan;<br />    public void tampilkanInfo() {<br />        System.out.println("Nama Kendaraan: " + nama);<br />        System.out.println("Kecepatan: " + kecepatan + " km/jam");<br />    }<br />}</pre><h4>Kelas Menengah: KendaraanDarat.java</h4><pre>public System.out.println("Bahan Bakar: " + bahanBakar);<br />    }<br />}</pre><h4>Kelas Turunan: Mobil.java</h4><pre>public class Mobil extends KendaraanDarat {<br />    int jumlahPintu;<br /> SepedaMotor.java</h4><pre>public class SepedaMotor extends KendaraanDarat {<br />    String jenisMesin;<br />    @Override<br />    public void tampilkanInfo() {<br />        super.tampilkanInfo();<br oleh semua kendaraan, yaitu nama dan kecepatan, serta metode tampilkanInfo() untuk menampilkan informasi dasar kendaraan.</li></ul><p><strong>Kelas Menengah </strong><strong>KendaraanDarat</strong>:</p><ul><li>Kelas KendaraanDarat merupakan turunan dari Kendaraan, yang lebih spesifik untuk kendaraan yang berjalan di darat.</li><li>Kelas ini menambahkan Dengan adanya <strong>kelas menengah</strong> seperti KendaraanDarat, kita bisa membuat <strong>pemodelan lebih modular</strong> dan fleksibel.</p><p>Pewarisan ini memudahkan kita dalam membuat berbagai jenis kendaraan dengan karakteristik umum, tanpa harus mendefinisikan ulang atribut dasar seperti nama dan kecepatan. Setiap kelas dapat fokus pada atribut khususnya sendiri, sementara karakteristik dasar tetap terpusat di kelas induk.</p><h3>Studi Kasus Tambahan: Pewarisan dengan Hewan</h3><p>Untuk memperdalam pemahaman tentang inheritance, kita akan mengimplementasikan pewarisan pada contoh lain. />}</pre><h4>Kelas Turunan: Kucing.java</h4><pre>public class Kucing extends Hewan {<br />    @Override<br />    public void tampilkanInfo() {<br />        super.tampilkanInfo();<br />      super.tampilkanInfo();<br />        System.out.println("Suara: Guk");<br />    }<br />}</pre><h4>Kelas Utama: Main.java</h4><pre>public class Main {<br />    public static void main(String[] kucing.tampilkanInfo();<br />        // Objek Anjing<br />        Anjing anjing = new Anjing();<br />        anjing.nama = "Rocky";<br />        anjing.jenis = "Anjing Bulldog";<br />        anjing.tampilkanInfo();<br />    }<br />}</pre><p><strong>Output:</strong></p><figure><img alt="" src="https://cdn-images-1.medium.com/max/827/1*R10tOwvOFkFn6rLsUuVxCQ.png" /></figure><p>Pada studi Struktur</h4><p><strong>Kelas Induk </strong><strong>Hewan</strong>:</p><ul><li>Kelas ini memiliki dua atribut, yaitu nama dan jenis, serta metode tampilkanInfo() yang bertugas menampilkan informasi
